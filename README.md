# SkillMapAgent

An AI-powered career guidance assistant that helps users analyze industry demand for skills and discover relevant job opportunities.

## Features

* Skill demand analysis
* Industry trend research
* Real-time job search
* Conversational memory
* Personalized career recommendations

## Tech Stack

* Python
* LangChain
* LangGraph
* Gemini 2.5 Flash
* Tavily Search API
* JSearch API

## How It Works

1. User enters a skill or career query.
2. Tavily Search researches industry demand and trends.
3. JSearch API finds relevant job opportunities.
4. Gemini analyzes the results.
5. The agent provides career insights and job recommendations.
